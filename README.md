# EnerVision 🌞🌬️

Renewable Energy Forecasting and Visualization Platform

## Features
- Solar energy generation forecasting
- Wind energy generation forecasting
- Confidence levels for predictions
- Forecast explanations
- Interactive dashboard
- Weather monitoring
- Kannada language support (Upcoming)

## Tech Stack
### Frontend
- React
- Vite
- Axios
- Chart.js

### Backend
- Node.js
- Express.js

### Database (Upcoming)
- MongoDB

### Machine Learning (Upcoming)
- Solar and Wind Forecasting Models
