# EnerVision 🌞🌬️

**Renewable Energy Forecasting and Visualization Platform**

EnerVision is a full-stack platform designed to predict and visualize solar and wind energy generation across Karnataka using weather monitoring, data analytics, and machine learning.

---

# 📌 Project Overview

The goal of EnerVision is to provide an interactive system that:

* Forecasts solar energy generation
* Forecasts wind energy generation
* Displays confidence levels for predictions
* Provides AI-based forecast insights
* Monitors weather conditions
* Visualizes data through interactive dashboards
* Supports Kannada language accessibility
